What is the Document Object Model (DOM)?

The DOM is a programming interface for web documents. DOM represents the structure of an HTML document as a tree of objects. you can Modify text, attributes, or styles of HTML elements.Dynamically create or delete elements with the dom.

How do you select an element from the DOM?

using getElementById() method you can select an specific element from HTML.

What is event delegation in the context of the DOM, and why is it useful?

Event delegation is a technique where a single event listener is attached to a parent element, allowing it to handle events from its children. This leverages event bubbling, improving performance by reducing the need for multiple listeners and supporting dynamic content added after the page loads.

How do you manipulate an element's attributes and styles using the DOM?

To manipulate DOM elements you can use getAttribute(), setAttribute(), or removeAttribute() for attributes, and style for inline styles or classList to add, remove, or toggle CSS classes.
